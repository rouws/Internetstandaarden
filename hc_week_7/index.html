Voorbeeld 1

Welke van de volgende stukken code is valide HTML5?

  1. <p><img src="plaatje.jpg" alt="plaatje" /></p>
  2. <p><img src="plaatje.jpg" alt="plaatje" />
  3. <p><img src="plaatje.jpg" alt="plaatje"></img></p>
  4. <p><img src="plaatje.jpg" /></p>

Voorbeeld 2

Vul de lege plekken in:
Een ___(I)___ link is een link die naar een pagina binnen jouw website verwijst.
Een ___(II)___ link is een link die naar een pagina buiten jouw website verwijst.

  1. I = interne, II = externe
  2. I = lokale, II = externe
  3. I = lokale, II = hyper
  4. I = interne, II = hyper

Voorbeeld 3

Bekijk onderstaande html-code en bedenk hoe de css-selector eruit ziet waarmee je de derde en vierde paragraaf kan vormgeven.

<p>Dit is de eerste paragraaf</p>
<p id="bijzonder">Dit is de tweede paragraaf</p>
<p class="afwijkend">Dit is de derde paragraaf</p>
<p class="afwijkend">En de vierde</p>

  1. p afwijkend
  2. p
  3. p#afwijkend
  4. p.afwijkend
