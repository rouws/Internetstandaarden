Huiswerkbespreking

  • Zijn er vragen over de oefeningen?
  • Is het gelukt met het opmaken van je artikel?
  • Straks controle van artikel
    • leesbaar?
    • correcte syntax?
    • verzorging van code?
    • semantisch correct?

Bedenk voorbeelden

  • Voorbeeld van een syntactische fout?
  • Voorbeeld van onverzorgde code?
  • Voorbeeld van semantische fout?

Vergelijk met elkaar

  • Is artikel goed leesbaar en begrijpelijk?
  • Is syntax correct?
    • Valideer je code met de W3C validator
      http://validator.w3.org/#validate_by_input
    • Valideer je code met Lint
      http://lint.brihten.com/html/#byinput
  • Is de code netjes verzorgd?
  • Is de code semantisch correct?
    Heb je de html-tags alleen gebruikt om de structuur van je content aan te geven?

Onze criteria

  • De HTML-code is semantisch correct.
  • De HTML-code is syntactisch correct.
  • De CSS-code is syntactisch correct.
  • De Javascript-code is syntactisch correct en wordt gebruikt voor DOM-manipulatie.
  • HTML is enkel gebruikt voor content, CSS enkel voor opmaak, JQuery enkel voor gedrag.
  • De website heeft een menu gebaseerd op unordered lists.
  • De website functioneert in Chrome en de layout is consistent doorgevoerd.
  • Zowel de website als de broncode zijn verzorgd en leesbaar opgemaakt.
  • De website bevat relevante inhoud in correct Nederlands.
  • De website staat op de in de les aangegeven server en is zonder technische problemen te bereiken.
  • De website bestaat uit tenminste vier pagina’s.
